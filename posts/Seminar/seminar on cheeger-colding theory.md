# About this Seminar
This Seminar will discuss Cheeger-Colding theory, which study the structure of Ricci limit 
space. Cheeger-Colding theory is heavily rely on some analysis tools, including classcial 
intergral inequality and some PDE's estimates. Although analysis tools are important, 
the most important part is using this tools to study the geometry i.e. analysis the structure of Ricci limit space.
