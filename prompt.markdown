# Role
你是一位全栈前端开发专家，擅长使用 HTML5, CSS3 (甚至 Tailwind CSS) 和 JavaScript 构建现代化、极简主义风格的个人网站。

# Task
参考 https://www.shinnku.com/ 的视觉风格，为我创建一个个人网站的初始界面。

# Requirements

1. 视觉风格：

- 极简主义：背景纯净（推荐浅色系或毛玻璃质感），字体排版优雅。

- 氛围感：参考参考网站的意境，使用大面积留白和精美的转场效果。

- 其他要求：请使用 Tailwind CSS 的 CDN 链接来编写样式，并加入一些简单的 Framer Motion 或原生 CSS 动画，让界面在加载时有淡入效果。

2. 顶部导航栏 (Navbar)：

- 包含以下菜单：随笔 (Essays)、个人简介 (About)、Demo (Projects)。

- 导航栏右侧需集成一个搜索框，用于关键字搜索，要求交互流畅。

3. 首屏布局 (Hero Section)：

- 中间放置一段具有设计感的欢迎语或个人 Slogan。

- 整体设计要响应式，适配手机和 PC 端。

4. 技术要求：

- 请提供一个单文件的 HTML（包含 CSS 和基础交互 JS），方便我直接预览。

- 使用现代化的布局方式（Flexbox 或 Grid）。

- 搜索功能暂做前端 UI 模拟，点击搜索图标或回车时能触发 console.log 提示。

# Output
请先简述你的设计思路，然后提供完整的代码。
